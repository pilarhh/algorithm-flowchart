# Algoritma

### Algoritma Reverse String

1. Mulai
2. Declare variabel str, newStr, dan i
3. Menginisiasi variabel newStr dengan string kosong
4. Input str
5. Menggunakan loop dengan index awal panjang string dikurangi 1 yang berkurang 1 setiap looping. Loop akan berhenti apabila i lebih dari sama dengan 0.
6. Menggunakan charAt untuk mengembalikan setiap huruf sesuai index loop yang setelahnya akan ditambahkan ke variabel newStr
7. Output newStr
8. Selesai

### Algoritma Deteksi Palindrom

1. Mulai
2. Declare variabel str, newStr, dan i
3. Menginisiasi variabel palindrom dengan string kosong
4. Input str
5. Menggunakan loop dengan index awal panjang string dikurangi 1 yang berkurang 1 setiap looping. Loop akan berhenti apabila i lebih dari sama dengan 0.
6. Menggunakan charAt untuk mengembalikan setiap huruf sesuai index loop yang setelahnya akan ditambahkan ke variabel palindrom
7. Menggunakan if else statement. Jika palindrom sama dengan str maka, True, output yang akan keluar berupa "Palindrom". Jika palindrom tidak sama dengan str maka, False, output yang akan keluar berupa "Bukan Palindrom"
9. Selesai

### Algoritma Mengubah Kata

1. Mulai
2. Declare variabel str, newStr, dan i
3. Menginisiasi variabel newStr dengan string kosong
4. Input str
5. Menggunakan loop dengan index awal 0 yang bertambah 1 setiap looping. Loop akan berhenti apabila i kurang dari sama dengan panjang string dikurangi 1.
6. Menggunakan if else statement dan menggunakan charAt untuk mengembalikan setiap huruf sesuai index loop yang setelahnya akan ditambahkan ke variabel newStr. Jika charAt pada index loop sama dengan huruf "a" maka True, akan berubah menjadi huruf "o". Jika bukan huruf "a" maka, False, tidak akan berubah. 
7. Output newStr
8. Selesai
